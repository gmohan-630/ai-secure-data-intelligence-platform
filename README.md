AI Secure Data Intelligence Platform

Overview:
This system analyzes logs and detects sensitive data and security risks.

Modules:
- Log Analyzer
- Detection Engine (Regex based)
- Risk Engine
- Insights Generator

Features:
- Detect emails, passwords, API keys
- Identify stack traces
- Generate risk score and level
- Provide insights

Tech Stack:
- Java
- HTML, CSS, JavaScript

How to Run:
1. Run LogAnalyzer.java
2. Open index.html
